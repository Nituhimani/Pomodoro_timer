# FocusOS – Pomodoro Productivity App
(https://pomodoro-timer-wheat-two.vercel.app/)


FocusOS is a modern, minimal productivity web application designed to help users stay focused while studying or working. It combines a Pomodoro timer, task management, customizable workspace, and music support to create a distraction-free focus environment.

---

## 🚀 Features

### ⏱ Pomodoro Timer

* 25-minute focus timer
* Start / Pause functionality
* Reset timer option
* Fullscreen focus mode

### 📝 Task Management

* Add tasks dynamically
* Mark tasks as completed
* Tasks appear as clean interactive cards
* Automatic **Productivity Score** calculation based on completed tasks

### 🎨 Workspace Customization

* Choose from **4 built-in wallpapers**
* Upload your own custom wallpaper
* Wallpaper is saved using browser local storage

### 🎵 Music Support

* Upload and play your own study music
* Built-in audio player appears automatically after upload

### 🖥 Fullscreen Focus Mode

* Toggle fullscreen mode
* Button dynamically changes to **Exit Fullscreen**

### ⚙ Settings Panel

* Access workspace settings via the **settings icon**
* Change wallpaper
* Upload music
* Clean modal interface

### 💾 Local Storage

* Saves wallpaper preference across sessions

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3 (Glassmorphism UI)**
* **Vanilla JavaScript**
* **LocalStorage API**
* **Fullscreen API**

No external frameworks or libraries were used, making the application lightweight and easy to understand.

## 🌐 Deployment

This project can be easily deployed using **Vercel**.

### Deploy Steps

1. Install Vercel CLI

```
npm install -g vercel
```

2. Navigate to the project folder

```
cd focusos
```

3. Deploy

```
vercel
```

Your app will be live at:

```
https://your-project-name.vercel.app
```
